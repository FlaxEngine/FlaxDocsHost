    Network API

    Flax implements a low-level networking interface with Berkeley sockets (cross-platform). It's exposed to scripting API and can be used in C++, C#, or Visual Scripting.

    Here is a server-client example using low-level network.

    API

    • Network - networking interface (platform-dependant implementation)
    • NetworkSocket - socket object (value structure)
    • NetworkEndPoint - network endpoint created from address, port and protocol
    • NetworkSocketOption - network socket options
    • NetworkSocketGroup - socket objects group for batch operations
